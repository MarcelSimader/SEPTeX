TikZNode¶

class TikZNode(coordinate=Point(x=0, y=0, relative=False), name='', label='', relative_to=None, style=TikZStyle(draw=True))¶

TikZNode represents a standard TikZ node. It holds information about its coordinate, name, label and style. A node must first be registered with a TikZPicture instance by writing it, which happens implicitly if not stated explicitly.

Comparisons of node objects will compare the names of each node.

>>> assert TikZNode(Point(5, 2), name="x0") == TikZNode(Point(20, -4), name="x0")
>>> assert TikZNode(Point(0, 0), name="origin") != TikZNode(Point(0, 0), name="o")
Parameters

  • coordinate (Point) – the coordinate at which this node should be placed

  • name (AnyStr) – the name which will be used to reference this node

  • label (AnyStr) – the label which will be displayed on the document for this node

  • relative_to (Optional[TikZNode]) – the node to consider coordinate to be relative to

  • style (TikZStyle) – the style to apply to this node

The command to draw the node in the document on its own. This is the standard when the node is written directly to a TikZPicture instance. This method is equivalent to __str__().

TikZPath¶

class TikZPath(coordinates, cycle=False, label=None, style=TikZStyle(draw=True))¶

TikZPath represents a collection of coordinates and nodes drawn using the \draw command of TikZ.

Parameters

  • coordinates (Collection[Union[Point, TikZNode]]) – a collection of Point or TikZNode objects for the path

  • cycle (bool) – whether or not to end the final path string with cycle, which will join the ending to the beginning

  • style (TikZStyle) – the style to apply to this path

property arrow_type: SEPTeX.TikZBase.TikZArrow¶
Returns

the arrow type of a regular path is always TikZArrow.LINE

property coordinates_string: str¶

Converts all of the coordinates of this instance to a string. If there are no coordinates then the empty string will be returned.

TikZDirectedPath¶

class TikZDirectedPath(coordinates, cycle=False, label=None, style=TikZStyle(draw=True), arrow_type=TikZArrow(key='-'))¶

TikZDirectedPath represents a path similarly to TikZPath but with additional information on the arrow type used to join up the coordinates.

Parameters

  • coordinates (Collection[Union[Point, TikZNode]]) – a collection of Point or TikZNode objects for the path

  • cycle (bool) – whether or not to end the final path string with cycle, which will join the ending to the beginning

  • style (TikZStyle) – the style to apply to this path

  • arrow_type (TikZArrow) – the TikZArrow to use for the arrow tip of the edge

property arrow_type: SEPTeX.TikZBase.TikZArrow¶
Returns

which arrow head type to use at the end of this path
